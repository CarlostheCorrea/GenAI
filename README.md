# GenAI
Carlos Correa's Gen AI Class Folder

Project 1:
[AI Grader](https://github.com/CarlostheCorrea/GraderProject)

GraderProject is a local FastAPI web application that helps evaluate written work using customizable rubrics. It supports document upload, rubric-based scoring, grammar and clarity feedback, follow-up Q&A, essay rewriting, and live fact-checking using OpenAI web search. The project also includes structured API endpoints, model routing, calibration examples, and validation workflows to improve grading reliability.

Project 2:
[GitHub Research Assistant](https://github.com/CarlostheCorrea/GitHubResearchAssistant)

GitHubResearchAssistant is a FastAPI-based AI tool that helps users understand unfamiliar GitHub repositories. It ingests a public repo, chunks the code intelligently, builds a local vector index, and uses hybrid search with RAG to answer questions with grounded source snippets and citations. The project also includes GraphRAG-style repository context, version comparison, onboarding guides, LLM-as-a-Judge review, and an interactive dependency map for exploring code structure.

Project 3:
[Research Atlas](https://github.com/CarlostheCorrea/ResearchAtlas)

ResearchAtlas is a FastAPI-based AI research assistant for exploring academic papers from arXiv. It searches and ranks papers, downloads and analyzes full PDFs, generates structured summaries, and lets users ask grounded follow-up questions with citations and evidence. The project uses LangGraph, MCP tools, ChromaDB vector search, LLM-as-a-Judge evaluation, Arize Phoenix tracing, and a persistent research library with ratings that improve future paper ranking.

Project 3:
[FilingLens](https://github.com/CarlostheCorrea/FilingLens)

FilingLens is a FastAPI-based SEC filing intelligence workspace for analyzing public companies through EDGAR filings. It uses RAG, a custom MCP server, ChromaDB retrieval, LangGraph multi-agent workflows, and human-in-the-loop review to generate filing-backed research answers, company comparisons, disclosure-change analysis, market gap discovery, and structured financial data insights. The project also includes XBRL financial extraction, claim-level verification, LLM-as-a-Judge evaluation, RAGAS scoring, prompt-injection protection, and optional local Ollama classifiers.
